# Stopify - Disseny Responsive i Adaptative

**Projecte desenvolupat per:**
* Pol Gonzalez
* Gerard Fornes

## Descripció del Projecte

Aquesta aplicació Android simula el flux d'autenticació (Login i Registre) d'una plataforma de música anomenada **"Stopify"**. 

L'objectiu principal d'aquesta pràctica és demostrar l'ús de **Jetpack Compose** per crear interfícies que siguin:
1.  **Responsive:** Els elements s'adapten fluidament a l'espai disponible.
2.  **Adaptative:** La disposició (layout) canvia dràsticament segons si el dispositiu és un mòbil (Compact) o una Tablet/Escriptori (Medium/Expanded).
